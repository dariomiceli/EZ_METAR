About


METAR simply delivered.

NYC METAR is a project of NYC-based software developer and hobbyist pilot Dario Miceli. It is intended to help pilots view METAR information quickly for airports surrounding the greater-NYC area. NYC METAR is built on Python and Flask and uses the Aviation Weather REST API for up-to-date information.
Special thanks to Tim and Corey for all their help.
